# MSDS6306 Unit 4 Homework

## Deliverables
The following documents are deliverables for Unit4:

#### MSDS6306_Assignment4_Dickens.rmd
Responses to the questions and tasks assigned in the unit 4 homework.  This is a combination of R code and responses to questions as defined by the individual questions or sub-questions.

#### MSDS6306_Assignment4_Dickens.html
Output of the .rmd file above.  Included is all R code and R code outputs, excepting where they would prove distracting (such as opening websites), or superfluous for an output file (installing packages).

#### fivethirtyeight.csv
The Dataset of College Recent Grads found in the fivethirtyeight dataset package.

#### Dickens_Assignment4_Swirl.txt
Contains all relevant swirl library output for the assigned units of swirl.
